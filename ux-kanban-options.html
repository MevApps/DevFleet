Kanban Grouping:
Option A: Each goal gets its own horizontal swimlane with phase columns inside it. Goals are visually isolated — clear hierarchy, but uses vertical space fast.
Option B: Single set of phase columns (like today), but tasks are grouped under collapsible goal headers within each column. More compact, still shows which tasks belong to which goal.
Option C: Kanban stays flat like today. Goal context shown via colored tags on each card. Pane 1 filter chips let you isolate a single goal. Stream view handles hierarchy; kanban stays phase-focused.
